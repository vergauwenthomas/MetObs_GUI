# MetObs_GUI
A Graphical interface of the MetObs_toolkit

# Install
Unstable: 
`pip install git+https://github.com/vergauwenthomas/MetObs_GUI.git`

Prerelease:
`pip install MetObs-GUI==0.0.0a1`

# Usage
In python:

```
import metobs_gui 
metobs_gui.launch_gui()
```
